Ejemplo - Carrito de Dominios

Información del Carrito

Este ejemplo muestra cómo agregar dominios al carrito y actualizar el contador.

Para usar en tu sitio, añade los atributos data-domain, data-tld y data-product-id a tus botones.

Funciones Disponibles

  • addToCart() - Agregar item simple al carrito
  • addDomainToCart(domainName, tld, productId) - Agregar dominio específico
  • clearCart() - Limpiar el carrito

Atributos de Datos

Para que los botones funcionen automáticamente, añade estos atributos:

  • data-domain="nombre" - Nombre del dominio (sin TLD)
  • data-tld="com" - Extensión del dominio
  • data-product-id="40" - ID del producto en el sistema
